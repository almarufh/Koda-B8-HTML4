# Form Perokok (Google Form Clone)

## Fitur Utama
1. Desain Mirip Google Form

* **Desain Mirip Google Form**: Tampilan modern dengan aksen warna khas dan tata letak berbasis kartu.
* **Tata Letak Responsif**: Menggunakan kombinasi Flexbox dan Grid dasar agar rapi di semua ukuran layar.
* **Struktur Tanpa Tag Form & Label**: Komponen dibangun murni menggunakan elemen `div` dan kelas khusus sesuai kebutuhan spesifik.
* **CSS Eksternal Terpisah**: Pengelolaan gaya tampilan yang rapi dan terisolasi dari file HTML.
* **Elemen Masukan Interaktif**: Menyediakan input teks dengan efek garis bawah dinamis, pilihan radio, dan kotak centang (checkbox).
